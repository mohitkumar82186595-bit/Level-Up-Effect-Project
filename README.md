# Level-Up-Effect-Project
Excited to share my latest frontend project — Level Up Effect Animation 🎮

In this project, I created a game-inspired Level Up system using pure HTML, CSS, and JavaScript. The project simulates an XP progression system where users gain experience points, fill up a progress bar, and trigger a visually engaging Level Up animation with glowing effects and particle explosions.

✨ Key Features:
✅ Dynamic XP Progress Bar
✅ Level Counter System
✅ Level Up Popup Animation
✅ Golden Glow Effects
✅ Particle Burst Animation
✅ Responsive Gaming UI
✅ Pure HTML, CSS & JavaScript

This project helped me practice DOM Manipulation, CSS Animations, JavaScript Events, and Interactive UI Design.

💡 Building small projects like this is a great way to strengthen frontend development skills and create engaging user experiences.
